# 포켓몬 카드 가격 계산기

스프레드시트로 관리하는 포켓몬 카드 가격 데이터를 기반으로, 보유 카드를 일련번호로 추가하고 총액을 계산하는 정적 웹앱입니다.

## 사용법

1. `index.html`을 브라우저에서 엽니다. (별도 빌드/서버 불필요)
2. **가격 데이터** 섹션에서 다음 중 하나를 사용해 데이터를 등록합니다.
   - `CSV 파일 업로드`: `번호`, `카드 이름`, `가격` 열이 있는 CSV/TSV 파일을 업로드.
   - `스프레드시트에서 붙여넣기`: 구글 스프레드시트에서 헤더와 함께 복사한 후 붙여넣기.
   - `샘플 데이터 보기`: 예시 데이터를 미리 채워줍니다.
3. **카드 추가** 섹션에 일련번호(예: `SV10 075`)를 입력하면 자동으로 카탈로그에서 매칭되어 보유 목록에 추가됩니다.
4. **내 카드 목록**에서 수량 조정, 검색, CSV 내보내기를 할 수 있습니다.

## 데이터 형식

스프레드시트 열 예시:

| 순번 | 번호 | 카드 이름 | 가격(원) |
|------|------|-----------|----------|
| 1 | sv10 001 | 피콘 | 300 |
| 2 | sv10 002 | 버섯꼬 | 300 |

- 일련번호 매칭은 대소문자/공백을 무시합니다 (`SV10 075` ≡ `sv10 075` ≡ `sv10  075`).
- 가격은 숫자만 추출됩니다 (`3,500원` → `3500`).

## 저장

가격 데이터와 보유 목록은 브라우저의 `localStorage`에 저장됩니다. 같은 브라우저에서 다시 열면 이어서 사용할 수 있습니다.

## 파일 구성

- `index.html` — UI 마크업
- `styles.css` — 스타일
- `app.js` — 모든 로직 (CSV 파싱, 카탈로그/보유 목록 관리, 저장/내보내기)
